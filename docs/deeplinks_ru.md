# Deeplink RU | [Deeplink EN](https://github.com/shareui/packit/blob/main/docs/deeplinks_en.md)

Deeplink-ссылки с префиксом `tg://packit` позволяют быстро выполнять действия внутри экосистемы Packit: открывать меню, управлять репозиториями, устанавливать и обновлять плагины. Здесь вся информация про них

## Формат ссылкок

`tg://packit?<команда>&<параметр1>=<значение1>&<параметр2>=<значение2>...`

- **команда** — основное действие.
- **параметры** — дополнительные уточнения.

---

## Доступные команды

### Меню и навигация
| Deeplink | Назначение |
|----------|------------|
| `tg://packit?settings` | Открыть настройки |
| `tg://packit?deeplink` | Информация о deeplink |
| `tg://packit?other` | Прочие функции |
| `tg://packit?contributors` | Список создателей |
| `tg://packit?docs` | Документация |
| `tg://packit?repo` | Меню репозиториев |
| `tg://packit?install` | Меню установки плагинов / выбора репозиториев |
| `tg://packit?update` | Меню обновления / выбор репозиториев |

### Репозитории
| Deeplink | Назначение | Переменные |
|----------|------------|------------|
| `tg://packit?repo` | Показать список репозиториев | --- |
| `tg://packit?repo=all` | Показать все доступные репозитории | --- |
| `tg://packit?repo=add&name=NAME&link=URL&icon=URL` | Добавить новый репозиторий (требуются параметры `name`, `link`, `icon`) | `NAME`, `LINK`, `ICON` |

### Установка плагинов
| Deeplink | Назначение | Переменные |
|----------|------------|------------|
| `tg://packit?install` | Открыть меню установки | --- |
| `tg://packit?install&repo=REPO` | Установить плагины из указанного репозитория | `REPO` |
| `tg://packit?install&repo=REPO&plugin=PLUGIN` | Установить конкретный плагин | `REPO`, `PLUGIN` |
| `tg://packit?install&repo=REPO&plugin=PLUGIN&mode=MODE` | Установить плагин в указанном режиме. <br> `mode=share` — поделиться ссылкой на плагин; `mode=private` — личная установка. | `REPO`, `PLUGIN`, `MODE` |

### Обновление
| Deeplink | Назначение | Переменные |
|----------|------------|------------|
| `tg://packit?update` | Обновить информацию о плагинах во ВСЕХ репозиториях | --- |
| `tg://packit?update&repo=REPO` | Обновить информацию о плагинах в репозитории REPO | `REPO` |

### Другое
| Deeplink | Назначение |
|----------|------------|
| `tg://packit?forum` | Форум сообщества |
| `tg://packit?problems` | Сообщить о проблеме |
| `tg://packit?pkill` | Закрыть приложение |

---

## Значения

Значение примеров значений в ссылках.

| Параметр | Описание |
|----------|----------|
| `REPO`   | Название репозитория |
| `PLUGIN` | ID плагина |
| `MODE`   | Режим установки (`share`) |
| `NAME`   | Название репозитория |
| `LINK`   | Ссылка на репозиторий (URL) |
| `ICON`   | Иконка репозитория (pack/id) |
