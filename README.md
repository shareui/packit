# PackIt • Official Repository

Service repository for PackIt operations, plugin catalog, and icon storage.

## What is PackIt?

PackIt is a plugin and resource catalog for [exteraGram](https://t.me/exteraGram), and also the **#1 largest plugin** among all exteraGram/AyuGram plugins.

## Repository contents

- Plugin catalog entries
- Icons for the catalog
- Service files for PackIt internal operations
